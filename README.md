**📊 SQL Data Analysis Project**

This repository contains my SQL Data Analysis Project, where I explored, analyzed, and extracted insights from structured datasets using SQL queries. The project focuses on strengthening core SQL concepts that are essential for Data Analyst roles, especially at the fresher level.

🔍 Project Overview

In this project, I worked with multiple relational tables and performed real-world analytical tasks such as:

Understanding and exploring the dataset

Writing SQL queries to answer business-oriented questions

Applying filters, aggregations, joins, and subqueries

Analyzing patterns, trends, and summaries from data The goal of this project is to demonstrate my SQL querying skills and my ability to think analytically using data.

🛠️ Skills & Concepts Used

SQL Basics (SELECT, WHERE, ORDER BY)

Aggregate Functions (COUNT, SUM, AVG, MIN, MAX)

GROUP BY & HAVING

JOINS (INNER, LEFT)

Subqueries

Pattern Matching (LIKE)

Date & conditional filtering

Data summarization and insights

🎯 Key Learnings

How to approach analytical questions using SQL
Writing optimized and readable SQL queries
Translating raw data into meaningful insights
Building confidence for SQL interviews and projects
